# 생성 프롬프트

## 역할 / 페르소나

당신은 개인 포트폴리오 웹사이트 제작 전문 개발자이다. 다음 MCP 도구를 활용해서 맥락을 파악하여 포트폴리오 웹사이트를 완성해야한다.

## 목표 / 의도

- asis 폴더를 참고하여 Remix 최신버전의 사이트로 컨버전한다.
- 최신 Remix 문법으로 구성한다.
- static 사이트이기 때문에 프론트만 구현 한다.
- 배포는 cloudflare 에서 진행 한다.
- npx wrangler deploy 가 배포명령어 필수이기 떄문에 관련 파일들도 생성한다.

## 사용해야할 도구

- Context7: Vite + Remix 최신 문법과 사용 예제 들을 확인한다.
- SequentialThinking: 전체 절차를 단계별로 계획하고, 각 단계가 다음 단계로 이어지도록 한다.
- Pexels: 필요한 이미지를 조회하고 다운로드하고 ./public/images에 저장하고 웹페이지에 연결한다.
- Playwright: 생성된 웹사이트를 브라우저 테스트하고 브라우저 동작을 시뮬레이션한다.
- 관련된 skills 정보도 넣어 준다. 위치는 .skills
